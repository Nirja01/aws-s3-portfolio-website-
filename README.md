# 🌐 AWS S3 Portfolio Website

## Project Overview
A personal portfolio website hosted entirely on AWS S3 using static website hosting.

## What I Built
- Created an S3 bucket with static website hosting enabled
- Configured bucket policies for public access
- Deployed HTML/CSS portfolio website
- Website accessible via AWS S3 endpoint URL

## Technologies Used
- AWS S3
- AWS IAM (Bucket Policies)
- HTML/CSS

## Live Website
[Click here to view live website](http://nirja-portfolio-website.s3-website.us-east-2.amazonaws.com)

## What I Learned
- Cloud object storage concepts
- S3 bucket configuration and policies
- Static website hosting on AWS
- Public access management
